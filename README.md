# Gym Homepage Design - Lab Bravo & Lab Charlie
This repository contains the homepage design for the Gym fictional company for the Module 2 LabBravo for the spring semester 2026 of CIS 209.
 This project demonstrates the planning, layout design, and technical analysis of a fictional gym homepage. 
 The repository documents the structural wireframe and outlines the technologies required for future implementation.

## Technical Requirements

### Project Overview
This project contains the homepage design for a fictional Gym company created for Module 3 – LAB CHARLIE (Design and Technical Analysis) in CIS 209, Spring 2026.  

The purpose of this phase is to define the technical structure, tools, and requirements needed before full website implementation.

---

### Languages Used
- HTML5 (semantic page structure)
- CSS3 (styling and layout)
- JavaScript (optional for interactive components)

---

### Layout & Design Techniques
- CSS Flexbox (navigation bar and service layout)
- CSS Grid (content + promotional banner section)
- Media Queries (responsive design for mobile devices)
- Semantic HTML elements (`header`, `nav`, `section`, `footer`)

---

### Hosting Method
- GitHub Pages (static site hosting)

---

### Version Control & Project Management
- Git for version tracking
- GitHub repository for source management
- GitHub Issues used for Agile/Scrum-style user story tracking

---

### External Frameworks / Libraries
- No external frameworks required for initial implementation
- Bootstrap may be considered for responsive enhancements (optional)

---

### API Integration
- None required for this project

---

### Development Tools
- Visual Studio Code
- Chrome Developer Tools for debugging and testing
- GitHub for collaboration and documentation

---

### Browser Compatibility
The website should function properly on:
- Google Chrome (latest version)
- Microsoft Edge (latest version)
- Mozilla Firefox (latest version)
